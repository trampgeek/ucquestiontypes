RoboTurtle Functions

Commands (all global functions) are:
    {% if 'move' not in disabledfunctions|default([]) %}
  • move() - move one cell forward
    • {% endif %} {% if 'turn_left' not in disabledfunctions|default([]) %}
  • turn_left() - rotate 90 degrees counterclockwise
    • {% endif %} {% if 'turn_right' not in disabledfunctions|default([]) %}
  • turn_right() - rotate 90 degrees clockwise
    • {% endif %} {% if 'take' not in disabledfunctions|default([]) %}
  • take() - pick up an item from the current cell, place it in the inventory, and return its type name (e.g. 'star')
    • {% endif %} {% if 'put' not in disabledfunctions|default([]) %}
  • put() - place the least-recently-picked up object in the current cell and return its type name
    • {% endif %} {% if 'toss' not in disabledfunctions|default([]) %}
  • toss() - place the least-recently-picked up object in the cell directly in front of the current cell, and return its type name
    • {% endif %} {% if 'send_text' not in disabledfunctions|default([]) %}
  • send_text(s) - display the string s in the text bar below the grid (only usable if this test supports it)
    • {% endif %} {% if 'print_state' not in disabledfunctions|default([]) %}
  • print_state() - print robot position and heading.
    • {% endif %} {% if 'speed' not in disabledfunctions|default([]) %}
  • speed(n) - set speed to n where n in range 1 - 10 (slow to fast). n = 0 for max possible speed.
    • {% endif %}
State-checking boolean functions
    {% if 'at_goal' not in disabledfunctions|default([]) %}
  • at_goal() - true if the robot is at the specified goal position
    • {% endif %} {% if 'front_is_clear' not in disabledfunctions|default([]) %}
  • front_is_clear() - true if the robot is able to move forward
    • {% endif %} {% if 'right_is_clear' not in disabledfunctions|default([]) %}
  • right_is_clear() - true if the cell to the right is clear
    • {% endif %} {% if 'wall_in_front' not in disabledfunctions|default([]) %}
  • wall_in_front() - true if there's a wall in front
    • {% endif %} {% if 'wall_on_right' not in disabledfunctions|default([]) %}
  • wall_on_right() - true if there's a wall to the right
    • {% endif %} {% if 'object_here' not in disabledfunctions|default([]) %}
  • object_here() - true if there's an object in the current cell
    • {% endif %} {% if 'carries_object' not in disabledfunctions|default([]) %}
  • carries_object() - true if the inventory is non-empty
    • {% endif %} {% if 'is_facing_north' not in disabledfunctions|default([]) %}
  • is_facing_north() - true if the robot is facing north
    • {% endif %}
State-query functions
    {% if 'position' not in disabledfunctions|default([]) %}
  • position() - return the robot's current grid coordinates as an (x, y) tuple
    • {% endif %} {% if 'target_cell' not in disabledfunctions|default([]) %}
  • target_cell() - return the target cell's coordinates as an (x, y) tuple, or None if no target is defined
    • {% endif %} {% if 'inventory' not in disabledfunctions|default([]) %}
  • inventory() - return a list of the type names of items in the inventory, ordered by when they were picked up (most recently picked up last)
    • {% endif %} {% if 'facing' not in disabledfunctions|default([]) %}
  • facing() - return the robot's current heading as 'N', 'S', 'E', or 'W'
    • {% endif %}
